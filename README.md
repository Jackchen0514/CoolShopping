# CoolShopping
一个仿拉手团购的购物App，采用Bmob后台实现短信验证码注册、登录、收藏、订单管理、自动更新等功能，数据抓取自拉手团购

# Library
 - NoHttp（网络请求）
 - zxing（二维码扫描）
 - Fresco（图片加载）
 - Andfix（热修复）
 - PullToRefresh（下拉刷新）
 - 自定义广告轮播
 - 自定义ScrollView
 - 自定义字母索引侧滑栏
 - ShareSDK第三方登录、分享
